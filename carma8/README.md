Задача:
1. Реализовать интерфейс просмотра коротких видео с вертикальным листанием,
2. аналогичный TikTok / Instagram Reels, с поддержкой mobile и desktop.

Технические условия:
1. Использовать только vanilla JavaScript (без фреймворков)
2. Разрешены современные Web APIs (IntersectionObserver, scroll-snap и т.п.)

Видео для наполнения интерфейса - по ссылке (https://drive.google.com/drive/folders/1L5lsFtOUSaIFt0nzQgo7fbAezBc0nBh-)

Дизайн:
Визуальный стиль не является ключевым фактором оценки.
Достаточно минимального UI

Формат сдачи:
Репозиторий (GitHub)
Короткий README с инструкцией запуска и описанием решения

Важно: Мы будем оценивать качество и структуру кода, отзывчивость интерфейса, а
также эффективность использования ресурсов браузера в рамках данного
тестового задания.
